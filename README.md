# Windows10-Report

## ・仮想デスクトップに名前を付ける機能の追加
     複数の作業を並行して行ったり、たくさんの資料を参考にして作業したりするなど、
     複合的な仕事を効率的に行う上で便利な機能である仮想デスクトップ一つ一つに自身で名前を付けられるようになる。
     

#### 仮想デスクトップとは
     仮想デスクトップは、コンピューターの一種でデスクトップの数、あるいは領域の拡張を行う。
     または、この種のソフトウェアによって拡張されたデスクトップのこと。
